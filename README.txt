Ajay Ramesh Employability Portfolio - Merged Activities Version

Open index.html in a browser.

Changes made:
- Removed the separate Photo Gallery section.
- Merged photos, activity headings, dates, types, and summaries into one Activities section.
- Each activity card is clickable.
- Clicking a card opens an interactive modal with:
  - activity title
  - date
  - type
  - main evidence image
  - extra evidence photos where available
  - full event summary
- Online events show a screenshot placeholder until you add your CareerHub/Zoom screenshots.

To add online screenshots:
1. Put the screenshot file inside the assets folder.
2. In index.html, find the relevant activity card.
3. Add the image path inside data-images, for example:
   data-images="assets/linkedin_screenshot.png"
